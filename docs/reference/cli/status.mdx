# Status Command

The `status` command helps you check the current state of your Xata CLI and project.

## status {#status}

Get the status of the Xata CLI.

```bash
xata status
```

This command displays information about:
- CLI version
- Authentication status
- Current project
- Current branch
- Connection status

## Global Flags

- `-h, --help` - Print help information and exit
- `--json` - Output in JSON format
