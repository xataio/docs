The `version` command helps you check the versions of the Xata CLI and pgroll components.

## version {#version}

Get the version of the Xata CLI and pgroll.

```bash
xata version
```

This command displays:
- Xata CLI version
- pgroll version
- Build information

## Global Flags

- `-h, --help` - Print help information and exit
- `--json` - Output in JSON format
