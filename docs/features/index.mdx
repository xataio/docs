# Features

Xata provides a comprehensive set of features to help you build and manage your database:

- [Branching](/features/branching) - Create and manage database branches for development and testing
- [Schema Changes](/features/schema-changes) - Safely modify your database schema with zero downtime
- [Data Anonymization](/features/data-anonymization) - Protect sensitive data in development environments

Each feature is designed to make database development more efficient and secure.
