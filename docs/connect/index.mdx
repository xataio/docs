# Connect to Xata

Xata provides multiple ways to connect your applications to your database. Choose the method that best fits your development workflow:

- [Languages](/connect/languages) - Connect using your preferred programming language
- [Frameworks](/connect/frameworks) - Integration guides for popular web frameworks
- [ORMs](/connect/orms) - Use your favorite Object-Relational Mapping tools

Each connection method provides full access to Xata's features while maintaining the development experience you're familiar with.
