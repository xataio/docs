The `version` command helps you check the current version of your Xata CLI.

## version

Get the version of the Xata CLI.

```bash
xata version [--json] [-h|--help]
```
- `--json`: Output in JSON format (default: false)
- `-h, --help`: Print help information and exit

## Global Flags

- `-h, --help`
