title: Upgrade Command
description: Command for upgrading the Xata CLI
---

The `upgrade` command helps you update the Xata CLI to the latest version.

## upgrade

Upgrade the Xata CLI.

```bash
xata upgrade [--channel <channel>] [--version <version>] [--json] [-h|--help]
```
- `--channel`: The channel to upgrade from (dev|next|latest, default: latest)
- `--version`: The specific version to upgrade to (default: "")
- `--json`: Output in JSON format (default: false)
- `-h, --help`: Print help information and exit

## Global Flags

- `-h, --help` - Print help information and exit
