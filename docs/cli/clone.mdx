title: Clone Command

---

The `clone` command helps you create a copy of your Xata database.

## clone

Create a clone of your Xata database.

```bash
xata clone
```

This command:
- Creates a new database branch
- Copies all data from the source branch
- Sets up the new branch for development

## Global Flags

- `-h, --help` - Print help information and exit
- `--json` - Output in JSON format
