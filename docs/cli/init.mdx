title: Init Command
description: Command for initializing a Xata project
---

The `init` command helps you set up a new Xata project in your current directory.

## init

Initialize a new Xata project.

```bash
xata init
```

This command:
- Creates a new project configuration
- Sets up the project structure
- Configures your local environment

## Global Flags

- `-h, --help` - Print help information and exit
