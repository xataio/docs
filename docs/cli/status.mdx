---

The `status` command helps you check the current state of your Xata CLI configuration.

## status

Check the status of your Xata CLI configuration.

```bash
xata status
```

This command displays information about your current configuration, including:
- Authentication status
- Current project
- Current branch
- Environment settings

## Global Flags

- `-h, --help` - Print help information and exit
- `--json` - Output in JSON format
